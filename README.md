## TextUtils

- Word Counter
- Character Counter
- Remove extra Spaces
- convert UpperCase to LowerCase
- convert LowerCase to UpperCase

# Installation

- npm install

**In Project Directory run**
`npm run start`

### Technologies Used

- React
- Bootstrap
